## Who am I?
My name is **Salomon Valverde**.

I’m a Software Developer, creator, and tinkerer. I’m currently working with [XYZ] developing software, writing documentation, building developer tools, automating continuous integration pipelines, and optimizing API endpoints for various clients both mobile and web.

I sometimes describe myself as an ~~Full~~ Entire Stack Developer given the fact that I've worked professionally and personally on most of the stacks or teams a software company has. 
Naturally I started my career building Rich Internet Applications (RIAs) using ColdFusion /J2EE. Then organically I moved over to Mobile Applications and completely saturated myself in the Apple iOS SDK ecosystem. At that time I had built several native mobile apps for iDevices mostly for b2b.

After my mobile dev interests ran its course (or rather [Apple became Apple](https://techdriveplay.com/2025/01/13/why-apple-stopped-using-intel-chips/)), I coincidentally jumped into the DevOps / Infrastructure as Code space and began my recipe writing and deploying scripts using Chef and Ansible. I learned a lot in a short amount of time as I was able to manage, automate and deploy hundreds of nodes at an enterprise level in the Government space for the DoD. Can't say much on these projects but one could imagine the level of infrastructure needed to manage our nations GPS satellites.
So throughout all these different verticals in the stack I was always working on the web/app servers delivering up content or consuming via some sort of API service.
